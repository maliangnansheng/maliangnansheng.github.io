设置（重置）密码

输入Redis命令：会提示(error) NOAUTH Authentication required. 这是属于正常现象。

我们输入：auth 123456

再次输入Redis命令即可正常使用

Windows

  1. 找到Redis安装目录，打开redis.windows.conf和redis.windows-service.conf并找到# requirepass foobared这一行，在这一行下面增加一行requirepass 你的密码

  2. 保存，重启Redis服务
