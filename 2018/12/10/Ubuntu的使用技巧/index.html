  • xx is not in the sudoers file 问题解决

    首先利用whereis 命令查找sudoers配置文件的目录（默认会在/etc/sudoers)

    然后就可以利用vi编辑器来把用户添加到sudoers之中

    然后找到root ALL=(ALL) ALL 或者 root ALL=(ALL:ALL) ALL所在的位置，把所要添加的用户添加到文件之中
